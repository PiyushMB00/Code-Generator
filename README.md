# PiscesForge ♓🚀
A fast, interactive UI component and web page builder.

Forge UI at the speed of thought.

---

## ⚡ What is PiscesForge?

PiscesForge lets you build, customize, and export UI components and full web pages directly in the browser.

No setup. No frameworks required.

---

## 🔥 Core Features

### 🎨 Designer Suite
- Google Fonts search with live preview
- Unsplash image picker
- Theme palettes (Ocean, Cyber, Aurora, etc.)
- Visual presets (Glass, Neon, Minimal, etc.)
- Undo / Redo history

---

### 🏗️ Canvas Builder
- Stack multiple components into a page
- Reorder, delete, or reset layout
- Toggle between component and full-page view

---

### 💻 Code Generation
- Clean HTML, CSS, and JS output
- Tailwind conversion (auto-generated classes)
- React (JSX) conversion
- Built-in interactivity (modals, tabs, etc.)

---

### 📦 Export Options
- Download standalone HTML file
- Export full project (ZIP with CSS/JS)
- Copy code instantly

---

### 📱 Responsive Preview
- Desktop / Tablet / Mobile views
- Smooth viewport switching
- Shareable project links

---

## 🧩 Components Included

- Navigation: Navbar, Sidebar, Breadcrumbs  
- Layout: Hero, Cards, Sections, Footer  
- Forms: Inputs, Buttons, Selects  
- Interactive: Modals, Tabs, Accordions  
- Data: Tables, Stats, Ratings  
- Feedback: Alerts, Loaders, Progress  

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- Google Fonts API
- Unsplash Integration
- JSZip (export system)
